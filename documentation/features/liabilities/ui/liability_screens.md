# Debt Manager UI Screens

## What these files do
These screens provide the interface for the Debt Manager (Liability Tracker).

## Key Screens
- **`LiabilitiesListScreen`**: 
  - Lists all active debts with repayment progress indicators.
  - Shows the remaining balance for each debt.
  - Provides a summary of total remaining debt.
- **`AddEditLiabilityScreen`**:
  - A form to track new debts or edit existing ones.
  - Includes fields for loan type, interest rates, and amounts.
  - Includes a quick delete option for debt records.
