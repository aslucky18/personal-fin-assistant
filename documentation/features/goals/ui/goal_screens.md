# Goal UI Screens

## What these files do
These screens provide the visual interface for users to manage their financial savings goals.

## Key Screens
- **`GoalsListScreen`**: 
  - Displays all goals in a card-based list.
  - Shows progress bars for each goal.
  - Provides a button to add new goals.
- **`AddEditGoalScreen`**:
  - A form to enter goal details (name, target, current savings, deadline).
  - Handles both creating new goals and updating existing ones.
  - Includes a delete button for existing goals.
