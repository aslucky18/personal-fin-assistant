# Setup Account Screen

## What this file does
This screen ensures a smooth start for new users by requiring them to create at least one bank account before they can reach the main dashboard.

## Key Features
- **Mandatory Action**: Users cannot skip this screen if they have zero accounts.
- **Simplified Form**: Asks for Bank Name, Account Type, and Last 4 Digits.
- **Integration**: Works seamlessly with the `OnboardingScreen` flow.
